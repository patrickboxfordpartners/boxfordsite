The Situation

Sully's Marina Lounge opened in 1936. Nearly 90 years later, they had zero digital presence.

No website. No online ordering. No way for new Marina District residents to discover them. The owners needed a modern web presence that didn't compromise their authentic dive bar character. And they needed it fast.

What We Built

Custom Website

Dark theme, neon accents, Kelly green for Irish heritage. Responsive design that works perfectly on mobile and desktop. Built from scratch—no WordPress, no templates, no bloat.

E-Commerce Integration

Square-powered merchandise store. T-shirts, hats, pint glasses—order online, pick up at the bar. First revenue stream beyond foot traffic.

Live Instagram Feed

Automatically pulls current posts from @sullysmarinalounge. No manual updates required. Site stays current without the owners touching code.

Complete Technical Deployment

GitHub Pages hosting. Cloudflare DNS configuration. Custom domain setup. HTTPS enabled. Zero monthly costs. Zero maintenance overhead.

Technical Stack

Lightweight, modern, built to last. No frameworks you don't need. No dependencies that break. Just clean code that works.

HTML5 CSS3 JavaScript GitHub Pages Cloudflare Square API Instafeed.js

Why This Matters

GitHub Pages means zero hosting costs. Not $10/month. Not $50/month. Zero. The site runs on enterprise infrastructure for free.

No CMS means no maintenance. No plugins to update. No security patches. No compatibility issues. The site just works.

Square integration uses existing payment processor. No new merchant account. No additional fees. Leveraged what they already had.

Instafeed pulls content automatically. Instagram gets updated, site gets updated. Zero manual work required.

The 5-Day Sprint

Day 1

Discovery. Defined the dive bar aesthetic, confirmed Irish heritage requirements, established goals.

Day 2

Built the site structure. Dark theme, Kelly green accents, responsive layout. All custom code.

Day 3

Integrated Square e-commerce and Instagram feed. Wrote all copy. Built custom photo carousel.

Day 4

Complete technical deployment. GitHub Pages configuration, Cloudflare DNS setup, domain verification.

Day 5

Final testing, cross-browser checks, launch. Site went live at sullysmarinalounge.com with HTTPS.

The Result

A 90-year-old bar now has a complete digital presence that maintains its authentic character.

New Marina District residents can find Sully's online. The merchandise store creates a new revenue stream beyond foot traffic. Instagram integration keeps the site current automatically. Hours and location information are always accessible.

Most importantly: the site doesn't try to be something it's not. It's a dive bar website for a real dive bar. No corporate polish. No fake authenticity. Just honest representation of what makes the place special.
